# SpaceShip
Цель проекта - реализовать прототип мобильной игры на андроид.

![Model](https://github.com/AlexChichibabin/SpaceShip/blob/40e901a611bb8b762fe7ba74f9e3cc5f75b4b545/SSh_ShipSelection.jpg)

## Реализовано:
1. Можно управлять кораблем с помощью виртуального джойстика
2. Можно стрелять бластером или самонаводящимися ракетами (находит цель во время полета)
3. Корабль может подобрать улучшения, которые можно найти на уровне
4. Противники с простым ИИ
5. Возможность победы/поражения в уровне
6. Астероиды наносят кораблю урон. При уничтожении астероида он разлетается несколько маленьких, которые можно уничтожить полностью
7. Планеты притягивают к себе корабль игрока и астроиды
8. В меню есть выбор из 3 кораблей с разными характеристиками
9. Есть статистика игрока и возможность сделать ее ресет
10. Анимации, звуки и музыка. Параллакс эффект на фоне уровня
